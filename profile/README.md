## What is DeesseJS?

DeesseJS is a TypeScript-first ecosystem of libraries and tools for building production SaaS. Every package is MIT-licensed, end-to-end typed, and built to compose with the others. No lock-in, no proprietary runtimes, no magic — just solid, boring, modern choices.

- **Comprehensive foundations** — error handling, FP monads, ORM, RPC, admin UI, cloud
- **Modern tooling** — pnpm, Turborepo, Drizzle, better-auth, shadcn/ui
- **Standards-driven** — Standard Schema for validation, JSON-RPC 2.0 error codes, Protobuf-compatible wire formats
- **Open-source, self-hostable** — every library is MIT, including the cloud control plane
